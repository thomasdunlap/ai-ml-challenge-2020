Technologies

Attain’s solution uses best in class open source technologies to ensure both technical proficiency on the back end and ease of use on the front end. Those technologies include Natural Language Processing with BERT for predictions, Tesseract OCR for pulling clauses from documents, and Flask for building Python Web Apps.

Natural Language Processing with BERT

In 2018, Google announced and released Bidirectional Encoder Representations (BERT), which has transformed the NLP field. BERT, now used heavily in Google search engines, achieves extremely accurate results while allowing for parallelization and transfer learning.

Tesseract OCR

Tesseract OCR is an open source tool for Optical Character Recognition, which uses Long Short Term Memory Neural Networks (LSTMs) extract text from images in a machine readable text format.

Flask for Python Web Apps

Flask is a lightweight WSGI we use as the projects web application framework. It is designed to make getting started quick and easy, with the ability to scale up to complex applications. It began as a simple wrapper around Werkzeug and Jinja and has become one of the most popular Python web application frameworks.

Functionality and User Interface:

● The solution provides a web interface, which allows users to browse their computers to select a file. After the file is selected, they click a button to run the application

● Behind the scenes, Tesseract OCR will extract clauses from the document

● Python / BERT will run predictions. The predictions will display on the web interface for each clause as the predictions are made.
